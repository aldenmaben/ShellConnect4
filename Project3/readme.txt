This is a two player Connect4 game which can be played using only two buttons. 

Rules:
	To navigate: l(as in lemon) button
	To select: enter button

Objective: Is to get 4 in a row either by a diagonal, row or column. 

Citation can be found in the .py file

.exe can be found in 'dist' folder

Enjoy!
